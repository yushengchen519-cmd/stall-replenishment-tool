## 🏪 日常余量 vs 🏭 仓库库存

工具支持两个模式切换：

| | 🏪 日常余量 | 🏭 仓库库存 |
|---|---|---|
| 显示库存 | 摊位货架上的数量 | 家里/仓库的存量 |
| 隐藏列 | 状态/供应商/成本/小计/存放位置 | 全部显示 |
| 补货来源 | 从仓库调拨（内部） | 向供应商采购（外部） |
| 补货效果 | 货架↑ 仓库↓ | 仓库↑ |

数据自动保存在浏览器中，刷新不丢失。

## 📋 日常使用流程

### 每日补货
1. 打开 [在线工具](https://yushengchen519-cmd.github.io/stall-replenishment-tool/)
2. 在 **🏪 日常余量** 模式，勾选货架不足的 SKU
3. 点击「生成补货 PDF」→ 保存图片到手机
4. 系统自动从仓库扣减 → 货架增加

### 仓库进货
1. 切到 **🏭 仓库库存** 模式
2. 勾选仓库不足的 SKU
3. 点击「生成补货 PDF」→ 作为采购单
4. 系统自动增加仓库库存

### 定期同步 Excel
1. 点击页面顶部的 **📤 导出Excel**
2. 打开 Excel v4 的「商品目录」表
3. 粘贴导出的数据覆盖 4 列（货架库存/货架安全线/仓库库存/仓库安全线）

### 数据重置
- 点击 **🔄 重置数据** 恢复初始值

## 📊 Excel v4 新增列

商品目录新增 4 列：

| 列 | 含义 |
|---|---|
| 货架库存 | 摊位上当前数量 |
| 货架安全线 | 低于此值需从仓库调拨 |
| 仓库库存 | 家里/仓库当前存量 |
| 仓库安全线 | 低于此值需向供应商采购 |

公式：总库存 = 货架库存 + 仓库库存
